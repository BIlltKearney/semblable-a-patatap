Semblable a Patatap

Uses paper.js and howler.js libraries along with html and css to combine keypresses with animations, sounds, shapes, and colors to create a responsive Patatap-like experience. The sounds are from Neuronal-Synchrony https://github.com/jonobr1/Neuronal-Synchrony
